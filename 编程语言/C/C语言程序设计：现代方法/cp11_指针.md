# 11 指针

## 11.1 指针变量

C 规定每个指针变量只能指向一种特定类型的对象。

## 11.4 指针作为参数

一般来讲，如果变量很大想用指针来传，但是又不希望在函数里修改变量的值，则可以使用 const 来限定指针指向的对象

下面的例子中 p 是指向「常整数」的指针。如果

```c
void f(const int *p)
{
    *p = 0;       // 编译器会报错
}
```

## 11.5 指针作为返回值

永远不要返回指向自动局部变量的指针

```c
int *f(void)
{
    int i;
    return &i;
}
```
